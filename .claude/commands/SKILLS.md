# Hustle Project Skills

Custom slash commands for this project. Invoke any with `/<skill-name>` in Claude Code.

| Skill | Description |
|-------|-------------|
| `/app-initiate` | Pull the latest code from GitHub on the current branch, rebuild Docker images, and start all Hustle services. |
| `/app-rebuild` | Rebuild only the Docker images whose source files have changed, then restart those containers. Leaves unchanged services running untouched. |
| `/app-shutdown` | Gracefully stop all running Hustle services via Docker Compose. |
| `/git-push` | Commit all pending changes and push to GitHub on the current branch. |

---

*Auto-regenerated by `/git-push` before every commit. To add a skill: create `.claude/commands/<name>.md` — the first line of the file becomes the description in this table.*
